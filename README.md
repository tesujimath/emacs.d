# emacs.d

This is my from-scratch .emacs.d using [Elpaca](https://github.com/progfolio/elpaca).

Elpaca is a fabulous modern Emacs package manager, and getting going with a from-scratch configuration was much easier than I expected.  That said, this will never become a starter kit, so if you want to make use of what I've done here, fork the repo and own the whole thing.
